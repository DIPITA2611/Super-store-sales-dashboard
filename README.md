
# 📊 Superstore Sales Dashboard — Power BI

## 📌 Project Overview

This project presents an **interactive Power BI dashboard** analyzing US retail sales data from 2019 to 2022.
The goal is to extract business insights related to sales performance, profitability, and customer segments.

---

## 📈 Key Metrics

* **Total Sales:** $2.30M
* **Total Profit:** $286K
* **Total Orders:** 9,994
* **Profit Margin:** 12.47%
* **Top Category:** Technology ($836K)
* **Top State:** California ($457K)

---

## 🛠️ Dashboard Features

* KPI Cards for Sales, Profit, Orders, and Profit Margin
* Region Filter (West, East, Central, South) with dynamic interaction
* Sales by Category (Bar Chart)
* Sales & Profit Trend Over Time (Area Chart)
* Sales by Ship Mode (Bar Chart)
* Top 10 Sub-Categories by Sales
* Sales by State (Map Visualization)
* Sales by Customer Segment (Donut Chart)

---

## 📁 Dataset

Superstore USA Retail Dataset (2019–2022)

## 🎯 Key Insights

* Technology category contributes the highest sales
* California is the top-performing state
* Profit margin remains consistent around 12%
* Regional filtering helps identify performance variations across regions
